# 语义记忆（Semantic Memory）

> 从情景记忆中提炼的可复用知识

---

## 什么是语义记忆？

语义记忆是从具体经验（情景记忆）中提炼出的**通用规律、方法论、最佳实践**。

### 与情景记忆的区别

| 维度 | 情景记忆 | 语义记忆 |
|------|----------|----------|
| 内容 | 具体事件 | 通用规律 |
| 时间性 | 有明确时间线 | 无时间限制 |
| 可复用性 | 特定场景 | 多种场景 |
| 示例 | "2026-02-15 的标题转化率很高" | "高转化标题的 3 个公式" |

## 提炼标准

### 何时提炼为语义记忆？

**触发条件**（满足任一即可）：
1. **重复模式**：同一规律出现 ≥3 次
2. **重大项目**：完成重要项目后的总结
3. **主动复盘**：每月/季度复盘时发现的规律

### 提炼流程

```
1. 识别模式
   ↓
2. 分析原因（为什么有效/无效）
   ↓
3. 提炼规律（形成可复用的方法）
   ↓
4. 创建语义记忆文件
   ↓
5. 持续验证和更新
```

## 使用语义记忆

### 对于 AI

**读取时机**：
- 涉及内容创作时，读取相关方法论
- 例如：生成标题时 → 读取 `标题创作_方法论.md`

**使用方式**：
- 基于方法论提供建议
- 用案例库支撑建议
- 提醒用户更新数据

### 对于用户

**日常使用**：
- 创作前先查看相关方法论
- 用方法论指导创作
- 创作后追踪数据

**定期维护**：
- 每月复盘时更新案例库
- 发现新规律时补充内容
- 失效的方法及时删除

## 进化路径

```
情景记忆（日记/对话历史）
        ↓
    识别重复模式（≥3 次）
        ↓
语义记忆（方法论/最佳实践）
        ↓
    发现必须遵守的规则
        ↓
强制规则（自动化决策）
```
