# 内容生产工作流 · 设计 Spec

> 版本：1.0 | 日期：2026-04-17 | 状态：已批准

## 目标

构建一套半自动内容运营系统，支持 X.com 和小红书的持续运营。系统分三条路径：每日自动研究、计划内容完整流程、热点速评轻量流程。所有知识统一沉淀到 Wiki，废弃独立素材库。

---

## 架构概览

```
每日定时研究（自动）
       ↓
   Wiki 进化（选题信号 + 爆款对标）
       ↓
  早报 → 用户选题
       ↓
  ┌────┴────┐
计划内容   热点速评
（完整流程）（轻量流程）
       ↓
   发布 X / 小红书
       ↓
   Wiki 沉淀
```

---

## 路径 0：每日定时研究（自动）

**触发**：每天早上定时任务（`tools/daily.sh`）

**执行内容**：
1. 抓取 X timeline 爆款（关注账号高互动帖，用 `baoyu-danger-x-to-markdown` 或 `x-mastery-mentor` 场景E）
2. 抓取 HN / Reddit 热点（现有脚本）
3. 对标账号更新（定期抓标杆账号近期内容，存入 `wiki/创作者/`）
4. Wiki ingest：新信号写入 `wiki/选题/` 相关页面，更新 `wiki/index.md`，追加 `wiki/log.md`

**输出**：`05-选题研究/早报-YYYY-MM-DD.md`

早报格式：
```markdown
# 早报 YYYY-MM-DD

## Timeline 爆款 Top 3
| 帖子 | 互动 | Hook 类型 | 结构特点 |
|------|------|-----------|---------|
| ... | ... | ... | ... |

## 今日热点信号
- ...

## 推荐选题
- [ ] [计划] 选题A — 角度说明
- [ ] [速评] 选题B — 热点说明（时效：今日）
- [ ] [计划] 选题C — 角度说明
```

## 路径 1：计划内容（完整流程）

1. **选题确认** — 从早报或 `01-内容生产/选题管理/00-选题记录.md` 取
2. **Wiki query** — 查相关选题页、方法论页、`wiki/素材/`，提炼可用角度和金句
3. **爆款对标** — 从早报 timeline 爆款找结构最近的 1-2 篇，分析 Hook/节奏/CTA
4. **创作主稿** — 嵌入 3-5 个人工素材（案例/数据/个人经历），AI 占比 ≤30%
5. **x-mastery-mentor 审稿** — 四层诊断（算法层 + Hook层 + 内容层 + CTA层）
6. **多平台改写** — X 长文 / 小红书图文（按需公众号）
7. **配图** — `/document-illustrator`（长文）或 `/baoyu-xhs-images`（小红书）
8. **发布** — `/baoyu-post-to-x` + `/post-to-xhs`
9. **Wiki 沉淀** — 更新选题页「已产出内容」，提炼金句/框架存入 `wiki/素材/`，追加 `wiki/log.md`
10. **收尾** — 更新 `tasks/progress.md`，git commit

**Definition of Done**：
- [ ] 主稿保存到 `01-内容生产/02-制作中的选题/`
- [ ] x-mastery-mentor 四层全部通过
- [ ] 发布完成（X + 小红书）
- [ ] wiki/选题/ 相关页面已更新
- [ ] wiki/素材/ 已提炼本篇金句/框架
- [ ] wiki/log.md 已追加
- [ ] tasks/progress.md 已更新
- [ ] git commit 已完成

---

## 路径 2：热点速评（轻量流程）

**门槛检查**：跟「中国独立开发者 + AI 工具」定位相关吗？不相关跳过。

1. **写稿** — 直接写，速度优先，目标 0-1h 内完成
2. **x-mastery-mentor 快速审稿** — 只过算法层 + Hook层
3. **发布** — `/baoyu-post-to-x`，附图
4. **事后补 Wiki** — 当天内 ingest 进相关选题页，追加 `wiki/log.md`

## Wiki 架构（唯一知识底座）

废弃 `02-内容素材库/`，全部迁移进 wiki：

```
wiki/
├── index.md          # 目录索引
├── log.md            # 操作日志（append-only）
├── 选题/             # 话题演化、信号、已产出内容
├── 方法论/           # 标题、开头、爆款规律
├── 素材/             # 金句、案例、可复用框架（新增）
│   ├── 金句库.md
│   ├── 案例库.md
│   └── 框架库.md
├── 创作者/           # 对标账号研究
└── 概念/             # 核心理论框架
```

**迁移规则**：
- `02-内容素材库/金句库/` → `wiki/素材/金句库.md`
- `02-内容素材库/案例库/` → `wiki/素材/案例库.md`
- `02-内容素材库/核心概念库/` → `wiki/概念/`（已有，合并）
- `02-内容素材库/爆款文稿库/` → 内容移入各选题页「已产出内容」，目录废弃
