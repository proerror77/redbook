# Opus 4.7 发布 · X 长文

---

Anthropic 今天发布了 Claude Opus 4.7。

我扒了一遍基准数据，说几个真正值得关注的点。

**最大的跳跃在 agentic coding。**
SWE-bench Pro 从 53.4% 跳到 64.3%，涨了将近 11 个点。SWE-bench Verified 从 80.8% 到 87.6%。这不是小幅迭代，是一个台阶。对用 Claude Code 写代码的人来说，这个提升是能感受到的。

**视觉推理涨得更夸张。**
CharXiv Reasoning（无工具）从 69.1% 跳到 82.1%，涨了 13 个点。这意味着 Opus 4.7 理解图表、截图、技术文档的能力大幅提升。做数据分析、看论文、处理带图的任务，体验会明显不同。

**和 GPT-5.4 的对比没有绝对赢家。**
Agentic coding 上 Opus 4.7 赢了（64.3% vs 57.7%），Scaled tool use 也赢了（77.3% vs 68.1%）。但 agentic search（BrowseComp）输了，79.3% vs 89.3%。两个模型各有擅长的场景，不是简单的谁全面碾压谁。

**但基准测试里最让我好奇的是最右边那一列：Mythos Preview。**
几乎每一项都比 Opus 4.7 高出一截——agentic coding 93.9%，terminal coding 82.0%，visual reasoning 93.2%。这是 Anthropic 下一代旗舰的预告？还是内部实验模型？没有任何官方说明，但它出现在这张对比表里不是偶然。

Anthropic 的发布节奏越来越快了。4.6 到 4.7，agentic 能力跳了一个台阶。而 Mythos Preview 还在路上。
