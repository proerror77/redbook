# 标题创作方法论

> **类型**：语义记忆 - 方法论
> **领域**：内容创作
> **来源**：从过往创作经验中提炼
> **更新时间**：2026-02-15
> **应用场景**：小红书图文、公众号文章、X.com 推文

---

## 核心原则

1. **制造信息差**：让读者觉得「这个我不知道」
2. **引发好奇心**：让读者想「然后呢？」
3. **承诺价值**：让读者相信「看完有收获」

## 标题公式

### 公式一：数字 + 痛点 + 解决方案
- 例：「3个让你效率翻倍的AI工具」

### 公式二：反常识 + 解释
- 例：「为什么高手从不追热点」

### 公式三：身份 + 行为 + 结果
- 例：「月入10万的博主都在用这个方法」

## 使用建议

**何时使用本方法论**：
- 为新内容设计标题时
- 优化已有内容标题时
- 分析爆款标题时

**使用流程**：
1. 根据内容类型选择标题公式
2. 生成 3 个标题选项
3. 评估每个标题的信息差/好奇心/价值承诺
4. 选择最优标题
5. 发布后追踪数据，更新案例库
