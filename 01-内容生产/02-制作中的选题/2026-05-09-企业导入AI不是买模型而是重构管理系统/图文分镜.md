# 图文分镜

## 核心主张

> 企业导入 AI 的第一步不是采购模型，而是把少数高价值流程改造成 AI 能理解、能执行、能被约束、能被复盘的管理系统。

## 视觉系统

- Template: `gpt-image-2-style-library` / `infographic-engine`
- Base broth: 克制、简洁、elegant，企业管理报告质感，留白充足，移动端可读，不使用泛紫蓝发光、机器人、bokeh、假 logo。
- Style seasoning: `enterprise operating blueprint`
  - Grid: Swiss grid + enterprise system map。
  - Palette: off-white background, graphite text, muted teal and amber accents。
  - Typography: editorial sans-serif feel, short readable labels only。
  - Annotation: thin connectors, numbered nodes, small audit trail markers。
  - Object relationship: model sits inside workflow, not above workflow。
- Shared avoid list: fake product UI, fake company logos, dense paragraphs, tiny random text, cyberpunk, glossy 3D robot, magic brain, cloud clipart。

## Layout QA

- [x] 每张图只有一个读者任务。
- [x] 统一 base broth 和一个主要 style seasoning。
- [x] X 图默认 16:9，不复用小红书 3:4。
- [x] 小红书每张卡绑定企业读者任务，不把长文段落硬塞进图。
- [x] 文字预算：主标题不超过 12 个中文，副标题不超过 18 个中文，标签最多 4-8 个。
- [x] 安全边距至少 8%。
- [x] 禁止随机小字、假 logo、假坐标、假品牌、密集段落。
