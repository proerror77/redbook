# 爆款规律

> 从已发布内容与高互动样本中提炼。
> 最后更新：2026-04-09

## 已验证的规律

### 对比框架

- `消费 vs 积累`
- `在业务里 vs 在业务上`
- 规律：两个对立状态 + 读者自我归类 + 结尾反问，天然适合 X 传播

### 外部信号开头

- 用 HN / Reddit / X 真实讨论量做开头
- 比直接说“这很重要”更有说服力

### 结尾金句

- `工具会被封，资产不会`
- `你现在是在业务里，还是在业务上？`
- 规律：短句、可复述、逼读者站队

### 新增：X 长文 / Thread 架构

- `X Article`：标题必须同时有现象和判断；开头前四段必须出现具体场景
- `X Thread`：第一条先抛判断，不做背景介绍；倒数第二条必须压成一句规则
- 详见：[[X-长文与Thread写作架构]]
