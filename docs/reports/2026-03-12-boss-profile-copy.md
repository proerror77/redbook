# BOSS 求职文案建议

日期：2026-03-12

## 1. 求职意向

优先改成：

- `AI Agent 架构师`

备选：

- `AI应用架构师`
- `智能体系统负责人`
- `企业 AI 落地负责人`

不要继续保留：

- `总裁/总经理/CEO`

## 2. 在线简历标题

推荐标题 1：

`AI Agent架构师｜Rust/Python｜工作流编排/高性能系统`

推荐标题 2：

`Agentic Engineer｜AI Agent / 自动化交付 / 高性能后端`

推荐标题 3：

`AI应用架构师｜多Agent协同｜Rust Python Node`

## 3. 个人优势

推荐版本：

`14年系统与业务经验，近年聚焦 AI Agent、工作流编排和高性能后端。熟悉 Rust、Python、Node.js，做过多 Agent 协同、自动化交付和低延迟系统，也能把复杂技术能力落到真实业务流程和企业智能化场景里。`

更偏企业落地版：

`主要做 AI Agent、自动化工作流和高性能系统工程，能把大模型能力融入真实业务流程。擅长 Rust/Python/Node.js、多 Agent 协同、工具调用和复杂系统交付，也有业务抽象和团队推进经验。`

## 4. 默认招呼语

推荐版本：

`你好，我这边主要做 AI Agent / 工作流编排 / 高性能系统，也有 Rust + Python 的落地经验。看了你的 JD，和我做过的智能体协同、自动化交付和复杂系统工程比较贴，方便继续沟通一下吗？`

更偏企业 AI 岗版本：

`你好，我主要做 AI Agent、自动化工作流和企业场景落地，也有 Rust/Python 的工程背景。看了你的 JD，和我做过的智能体系统、流程自动化和复杂后端比较贴，方便继续聊一下吗？`

## 5. 搜索关键词建议

优先保留：

- `AI Agent`
- `智能体`
- `RAG`
- `工作流`
- `AI应用架构师`
- `AI架构师`
- `技术负责人`
- `平台架构`
- `infra`
- `企业AI`
- `Agent infra`
- `知识库`

优先排除：

- `实习`
- `兼职`
- `合伙人`
- `训练师`
- `标注`
- `Power Platform`
- `机器人`
- `TTS`
- `ASR`
- `C++`

## 6. 回复优先级

优先回复：

- `什么内容`
- `好的 谢谢`
- `明白`
- `已发附件简历`
- `方便沟通 / 面试 / 电话`

不再重复投：

- 明确出现 `不合适 / 不太合适 / 暂不匹配 / 已招满 / 未通过`
