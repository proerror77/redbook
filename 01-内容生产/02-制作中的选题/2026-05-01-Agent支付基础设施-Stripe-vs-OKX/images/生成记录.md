# 配图生成记录

## 目标

为 X.com Article 生成一张 `16:9` 主图：`Agent 支付的权限层`。

## 尝试 1：baoyu-image-gen / Tuzi

命令：

```bash
npx -y bun .agents/skills/baoyu-image-gen/scripts/main.ts --promptfiles '.../images/prompts/x-agent-payment-permission-layer.md' --image '.../images/x-agent-payment-permission-layer.png' --preset article-elegant --ar 16:9 --quality 2k --json
```

结果：

- 失败。
- Tuzi API 返回 `Invalid Token`。
- 当前环境只检测到 `TUZI_API_KEY` / `TUZI_IMAGE_MODEL`，没有其他可用图像 provider 环境变量。

## 兜底产物

- 使用本地 SVG 生成可发布概念主图。
