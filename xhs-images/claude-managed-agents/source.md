# Claude Managed Agents 发布

Anthropic 于 2026年4月8日正式发布 Claude Managed Agents（公测版），这是一套面向开发者和企业的 AI Agent 托管平台。

## 核心亮点

1. 开发周期从「几个月」压缩到「几天」
2. 自动处理容器化、状态管理、工具编排、凭证安全
3. 支持长时间运行（数小时）的自主 Agent
4. 多 Agent 协作，并行处理复杂任务
5. 内置权限管理、身份验证、执行追踪
6. 任务成功率提升最高 10 个百分点

## 定价

Claude 模型费用 + $0.08/Agent 运行小时

## 已有客户

Notion、Rakuten、Asana、Sentry 等

## 一句话总结

你只需定义任务、工具和规则，Anthropic 帮你跑基础设施。
