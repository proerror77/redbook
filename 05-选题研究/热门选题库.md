## 热门选题模板

### 模板一：方法论型
**标题公式**：数字 + 方法 + 结果
**示例**：「3个让你效率翻倍的AI工具」

**内容结构**：
1. 痛点引入（为什么需要这个方法）
2. 方法详解（具体怎么做）
3. 案例展示（实际效果）
4. 行动建议（如何开始）

### 模板二：反常识型
**标题公式**：反常识观点 + 解释
**示例**：「为什么高手从不追热点」

**内容结构**：
1. 常见认知（大多数人的做法）
2. 反常识观点（为什么这样做是错的）
3. 正确做法（应该怎么做）
4. 底层逻辑（为什么这样做有效）

### 模板三：案例拆解型
**标题公式**：身份 + 行为 + 结果
**示例**：「月入10万的博主都在用这个方法」

**内容结构**：
1. 案例展示（具体数据和结果）
2. 方法拆解（他是怎么做的）
3. 可复用要素（你可以学什么）
4. 行动建议（如何开始）
