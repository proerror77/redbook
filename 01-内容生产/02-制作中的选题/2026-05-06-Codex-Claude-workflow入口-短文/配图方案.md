# 配图方案

## 配图目标

- 平台：`X`
- 形式：`16:9 单图观点卡`
- 作用：不是解释所有信息，而是把核心判断视觉化
- 核心文案：`赢家不是模型，是入口`

## 视觉方向

- 方向：`简洁 editorial tech poster`
- 关键词：`workflow gate / repo / CI / review / issue / developer operating system`
- 气质：克制、锐利、判断感强，不做花哨赛博风

## 画面概念

- 左侧：抽象的“模型竞技场”，像漂浮的能力卡片或 benchmark 条形条，表示旧竞争层
- 右侧：真正发光的是一条开发工作流通道，依次出现 `Issue -> Repo -> CI -> Review -> Merge`
- 中央：两张窄而高的工具面板，分别暗示 `Codex` 与 `Claude Code`，但不做大 Logo 堆砌
- 视觉重心：不是谁更亮，而是谁站在 workflow 通道门口

## 生图 Prompt

### base broth

Create a clean editorial 16:9 opinion card for X about AI coding competition shifting from model capability to workflow entry points. Show a contrast between an old model-arena on the left and a real engineering workflow gate on the right. The right side should depict a clear pipeline of issue tracker, git repo, CI checks, code review, and merge flow. Two narrow developer-tool panels can hint at Codex and Claude Code competing to stand at the workflow entrance. Minimal, sharp, high signal, no clutter, no meme energy.

### style seasoning

- editorial technology poster
- restrained premium palette
- off-white paper background with charcoal, muted steel, and one strong vermilion accent
- subtle UI panel glow, thin grid lines, mono labels
- modern newspaper meets developer ops dashboard
- not cyberpunk, not neon overload, not cartoon, not 3D toy

### content variables

- core metaphor: `the moat is the workflow gate, not the model arena`
- workflow labels visible but minimal: `Issue`, `Repo`, `CI`, `Review`, `Merge`
- optional small side labels: `context`, `permissions`, `audit`, `handoff`
- left side should feel noisy and commoditized
- right side should feel structured, owned, and operational
- keep product hints subtle, avoid oversized brand marks

### text budget

- title max 10 Chinese chars: `赢家不是模型，是入口`
- subtitle max 20 Chinese chars: `repo / CI / review / issue flow`
- no paragraph text inside image
- only 1 title + 1 subtitle + tiny workflow labels

### layout QA

- exact 16:9 composition
- safe margins for X mobile crop
- title must remain readable at thumbnail size
- no text overlapping with UI panels
- workflow labels should support the idea, not dominate the card
- maintain strong contrast between left “model arena” and right “workflow gate”
- no SVG look, no infographic overload, no watermark, no fake screenshots

## 图内文字建议

- 主标题：`赢家不是模型，是入口`
- 副标题：`repo / CI / review / issue flow`

## 使用提醒

- 本文件只输出 prompt，不实际调用生图。
- 如果后续真的生成，先检查中文标题是否被模型画错；必要时改成英文副标题 + 后期手工压字。
