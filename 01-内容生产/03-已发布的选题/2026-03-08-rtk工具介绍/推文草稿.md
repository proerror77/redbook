## 最终文案

最近看到一个很实用的开源工具 rtk。

它不去重新发明 AI IDE，只做一件事：在命令输出进入 LLM 上下文前，先过滤、压缩一遍。README 里的示例里，30 分钟 Claude Code 会话 token 大约从 11.8 万降到 2.39 万，差不多省了 80%。

Rust 单一二进制，零依赖。
