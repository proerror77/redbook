# 内容生产 Agent 思维

> 来源：`02-内容素材库/核心概念库/内容生产Agent思维.md` | 最后更新：2026-04-05

## 核心定义

内容生产 Agent 不是工具，是你的数字员工。它改变的不是效率，而是你的角色。

## 工具 vs 员工

| 维度 | 工具思维 | 员工思维 |
|------|---------|---------|
| 交互方式 | 告诉它怎么做 | 告诉它做什么 |
| 你的角色 | 操作员 | 管理者 |
| 时间花在 | 如何让 AI 听懂指令 | 判断结果好不好 |

## 关键洞察

Agent 训练好后，它知道：你的选题逻辑、素材库位置、写作风格、平台规则。
你要做的是：设定目标 → 验收结果 → 迭代优化。
