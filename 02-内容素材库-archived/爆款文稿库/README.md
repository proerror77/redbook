# 爆款文稿库

> 存放已验证的高质量文稿，供后续复用和参考

## 使用说明

### 什么样的文稿应该放入爆款文稿库？

1. **数据表现优秀**
   - 播放量/阅读量超过平均水平 2 倍以上
   - 互动率（点赞、评论、转发）高
   - 涨粉效果明显

2. **内容结构可复用**
   - 有清晰的内容框架
   - 逻辑结构可以套用到其他选题
   - 表达方式值得学习

3. **用户反馈好**
   - 评论区反馈积极
   - 有用户主动分享
   - 收藏率高

### 如何使用爆款文稿库？

1. **深化选题前先检索**
   - 看看是否有相关的爆款文稿
   - 分析其成功要素
   - 考虑是否可以复用框架

2. **提炼可复用要素**
   - 标题公式
   - 开头方式
   - 内容结构
   - 表达技巧

3. **持续迭代优化**
   - 根据新的数据表现
   - 更新爆款文稿库
   - 淘汰过时的内容

## 文稿命名规范

格式：`日期-平台-选题名称.md`

示例：`2024-01-15-小红书-AI如何提升工作效率.md`

## 文稿模板

每个爆款文稿应包含：
- 标题
- 正文内容
- 数据表现（播放量、点赞、评论等）
- 成功要素分析
- 可复用框架
