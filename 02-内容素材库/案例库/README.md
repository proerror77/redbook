# 案例库

> 收集和整理各类案例素材，用于内容创作时引用

## 案例分类

### 成功案例
- 个人成长案例
- 商业成功案例
- 内容创作案例
- AI 应用案例

### 失败案例
- 常见错误
- 踩坑经验
- 反面教材

## 案例收集标准

### 好案例的特征
1. **真实可信**：有具体数据和细节
2. **有代表性**：能说明普遍规律
3. **有启发性**：能给读者带来思考
4. **可验证**：有来源出处

## 使用说明

1. **随时收集**：看到好案例立即记录
2. **分类整理**：按主题分类存放
3. **定期更新**：补充新案例，淘汰过时案例
4. **创作时检索**：写文稿前先查找相关案例
